## Node.js 写的一个简易的博客后台
### 如何使用
1. clone git@github.com:coderlongren/nodejs-.git  
2. npm install express --save
3. 按照 package.json里面的配置npm install 所需模块
4. 安装 MongoDB，并启动
5. set DEBUG=blog:* & npm start // 限于Windows平台


主要使用了 Express， MongoDB
